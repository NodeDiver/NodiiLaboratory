# NodiiLaboratory

Laboratorio de aprendizaje práctico sobre Bitcoin, Lightning Network y Nostr.

Cada carpeta es un curso autocontenido con documentación paso a paso para que puedas replicar todo desde cero.

---

## Cursos Disponibles

| Curso | Descripción | Estado |
|-------|-------------|--------|
| [Bitcoin Node Setup](./bitcoin-node-setup/) | Montar un nodo Bitcoin completo desde cero en una VM | En progreso |
| [Nostr Wallet Connect](./nostr-wallet-connect/) | Implementar NWC para conectar wallets Lightning vía Nostr | Proximamente |
| [AI Agents + Bitcoin](./ai-agents-bitcoin/) | Agentes de IA que interactúan con la red Bitcoin/Lightning | Proximamente |
